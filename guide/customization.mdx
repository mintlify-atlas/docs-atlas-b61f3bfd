# Customization

Novelty provides extensive customization options to create your perfect reading environment. All settings are saved automatically and persist across app restarts.

## Accessing Settings

Most reading-related settings are in the Reader Settings page:

1. Tap **More** from the bottom navigation
2. Select **Reader Settings**
3. Adjust any settings
4. Changes apply immediately

## Text Display Settings

Customize how novel text appears.

### Font Size

Adjust the base size of all text:

<Steps>
  <Step title="Open Reader Settings">
    Navigate to **More** > **Reader Settings**.
  </Step>
  
  <Step title="Adjust Font Size Slider">
    Drag the **Font Size** slider left or right:
    - Minimum: 12pt (small)
    - Default: 16pt (medium)
    - Maximum: 24pt (extra large)
  </Step>
  
  <Step title="Preview Changes">
    Return to reading to see the new font size immediately.
  </Step>
</Steps>

**Recommended Sizes**:
- **Mobile**: 14-18pt
- **Tablet**: 16-20pt
- **Desktop**: 18-22pt

<Tip>
If text seems too large or small after adjusting, try changing the line height as well for better readability.
</Tip>

### Line Height

Control spacing between lines of text:

<Steps>
  <Step title="Open Reader Settings">
    Go to **More** > **Reader Settings**.
  </Step>
  
  <Step title="Adjust Line Height Slider">
    Drag the **Line Height** slider:
    - Minimum without ruby: 1.0x (very compact)
    - Minimum with ruby: 1.3x (prevents overlap)
    - Default: 1.5x (comfortable)
    - Maximum: 2.5x (very spacious)
  </Step>
  
  <Step title="See Changes">
    Return to reading to preview the spacing.
  </Step>
</Steps>

**Line Height Guide**:
- **1.0x-1.2x**: Compact, for experienced readers (ruby disabled)
- **1.3x-1.5x**: Balanced, suitable for most readers
- **1.6x-2.0x**: Spacious, easier on the eyes
- **2.0x+**: Very spacious, for accessibility

<Note>
When ruby (furigana) display is enabled, the minimum line height is automatically set to 1.3x to prevent text overlap. Disable ruby to use tighter spacing.
</Note>

### Font Family

Choose your preferred font:

<Steps>
  <Step title="Open Reader Settings">
    Navigate to **More** > **Reader Settings**.
  </Step>
  
  <Step title="Select Font">
    Choose from available fonts:
    - **Noto Sans JP** (default): Clean, modern sans-serif
    - Additional fonts may be available depending on your platform
  </Step>
  
  <Step title="Apply Font">
    The selected font applies immediately to all reading.
  </Step>
</Steps>

<Info>
Noto Sans JP is included with Novelty and designed specifically for Japanese text, providing excellent readability and comprehensive character support.
</Info>

### Ruby (Furigana) Display

Toggle pronunciation guides for kanji:

<Steps>
  <Step title="Open Reader Settings">
    Go to **More** > **Reader Settings**.
  </Step>
  
  <Step title="Toggle Display Ruby">
    Turn the **Display Ruby** toggle on or off:
    - **On** (default): Furigana appears above/beside kanji
    - **Off**: Only base text is shown
  </Step>
  
  <Step title="Adjust Line Height (Optional)">
    With ruby disabled, you can now use line height below 1.3x for more compact text.
  </Step>
</Steps>

**When to Disable Ruby**:
- You're a fluent Japanese reader
- You want more compact text
- You prefer cleaner appearance
- You're using very tight line spacing

**When to Enable Ruby**:
- You're learning Japanese
- You want pronunciation help for unfamiliar kanji
- You prefer authentic Japanese text appearance
- Line height is 1.3x or above

## Reading Direction Settings

Choose between vertical and horizontal text layout.

### Vertical Text (Tategaki)

Traditional Japanese vertical writing:

<Steps>
  <Step title="Enable Vertical Text">
    In **Reader Settings**, turn on the **Vertical Text** toggle.
  </Step>
  
  <Step title="Configure Vertical Reading">
    Consider adjusting:
    - Ruby appears to the right of characters
    - Swipe up/down to navigate episodes
    - Works best on portrait-oriented devices
  </Step>
</Steps>

**Vertical Text Advantages**:
- Authentic Japanese reading experience
- Natural for Japanese literature
- Immersive and traditional
- Better use of portrait screens

### Horizontal Text

Modern horizontal left-to-right layout:

<Steps>
  <Step title="Disable Vertical Text">
    In **Reader Settings**, turn off the **Vertical Text** toggle.
  </Step>
  
  <Step title="Configure Horizontal Reading">
    Consider adjusting:
    - Ruby appears above characters
    - Swipe left/right to navigate episodes
    - Works well on landscape-oriented devices
  </Step>
</Steps>

**Horizontal Text Advantages**:
- Familiar to readers of Western languages
- Better for wide screens (tablets, desktops)
- Easier to read on landscape orientation
- Faster for some readers

## Navigation Settings

Customize how you move through episodes.

### Page Flip Mode

Choose between scrolling and page flipping:

<Steps>
  <Step title="Open Reader Settings">
    Go to **More** > **Reader Settings**.
  </Step>
  
  <Step title="Toggle Page Flip Mode">
    Turn **Page Flip Mode** on or off:
    - **Off** (default): Continuous scrolling between episodes
    - **On**: Discrete page transitions
  </Step>
</Steps>

**Scrolling** (Page Flip Off):
- Smooth, continuous reading
- No interruptions between episodes
- Better for long reading sessions

**Page Flipping** (Page Flip On):
- Clear page boundaries
- Distinct episode transitions
- Feels more like reading a physical book

## Privacy Settings

Control what data Novelty tracks.

### Incognito Mode

Read without saving history:

<Steps>
  <Step title="Enable Incognito Mode">
    In **Reader Settings**, turn on the **Incognito Mode** toggle.
  </Step>
  
  <Step title="Read Normally">
    All reading you do while incognito mode is enabled won't be saved to history.
  </Step>
  
  <Step title="Disable to Resume Tracking">
    Turn off **Incognito Mode** when you want history tracking again.
  </Step>
</Steps>

<Warning>
Incognito mode prevents history tracking, meaning your reading position is NOT saved. If you close the app or navigate away, you'll need to manually find your place when you return.
</Warning>

**Use Incognito Mode When**:
- Re-reading novels you've already finished
- Sampling novels you might not continue
- Reading without wanting recommendations based on that content
- Privacy is a concern

## Appearance Settings

Customize the app's overall look.

### Theme Mode

Choose between light, dark, or system-based theme:

<Steps>
  <Step title="Open Appearance Settings">
    Go to **More** > **Appearance Settings**.
  </Step>
  
  <Step title="Select Theme Mode">
    Choose:
    - **System** (default): Follow device theme settings
    - **Light**: Always use light theme
    - **Dark**: Always use dark theme
  </Step>
</Steps>

**Light Theme**:
- Better for bright environments
- Reduces eye strain in daylight
- Traditional reading appearance

**Dark Theme**:
- Better for low-light environments
- Reduces blue light exposure
- Easier on eyes at night
- Saves battery on OLED screens

**System**:
- Automatically matches your device settings
- Switches with system theme changes
- Convenience of no manual switching

## Optimal Settings Presets

Recommended setting combinations for different scenarios.

### Mobile Portrait (Vertical Reading)

```
Vertical Text: On
Font Size: 16-18pt
Line Height: 1.5x
Ruby Display: On
Page Flip: Off
Theme: System
```

**Best For**: Traditional Japanese novel experience on smartphones

### Tablet Landscape (Horizontal Reading)

```
Vertical Text: Off
Font Size: 18-20pt
Line Height: 1.8x
Ruby Display: On
Page Flip: Off
Theme: System
```

**Best For**: Comfortable wide-screen reading on tablets

### Desktop (Large Screen)

```
Vertical Text: Off (or On, preference)
Font Size: 20-22pt
Line Height: 2.0x
Ruby Display: On
Page Flip: On
Theme: System
```

**Best For**: Distraction-free reading on desktop computers

### Compact/Fast Reading

```
Vertical Text: Off
Font Size: 14-16pt
Line Height: 1.2x
Ruby Display: Off
Page Flip: Off
Theme: Dark
```

**Best For**: Fluent readers who want maximum text density

### Accessibility (High Visibility)

```
Vertical Text: Off
Font Size: 22-24pt
Line Height: 2.0-2.5x
Ruby Display: On
Page Flip: On
Theme: Light (high contrast)
```

**Best For**: Readers who need larger text and more spacing

## Settings Persistence

All settings are saved automatically:

- **Immediate Application**: Changes take effect as soon as you adjust them
- **Persistent**: Settings survive app restarts and device reboots
- **Per-Device**: Settings are stored locally and don't sync across devices
- **Backup Included**: Settings are included in database backups

## Resetting Settings

To return to default settings:

1. Manually adjust each setting back to default values (listed above)
2. Or reinstall the app (also clears library and history)

There is currently no one-click "Reset to Defaults" button.

## Troubleshooting

### Settings Not Saving

- Ensure you have adequate storage space
- Check that the app has permission to write data
- Restart the app and verify settings persisted
- If persistent, reinstall the app (back up data first)

### Text Appearing Too Small/Large

- Adjust **Font Size** slider
- Check device accessibility settings aren't overriding app settings
- Ensure you're not zooming the screen accidentally

### Ruby Text Overlapping Lines

- Enable **Display Ruby** if currently disabled
- Increase **Line Height** to at least 1.3x
- Ensure you're not using an unsupported font

### Changes Not Visible

- Return to reading view to see changes
- Settings don't update in the settings page itself
- Try switching to a different episode and back
- Restart the app if changes don't appear
